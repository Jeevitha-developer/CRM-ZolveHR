# HRMS CRM

A comprehensive CRM system for managing HRMS subscriptions and client relationships.

## Features

- ✅ CRM Login System with HRMS integration
- ✅ Master entities for clients, subscriptions, and plans  
- ✅ Subscription status tracking with payment monitoring
- ✅ Three plan types: Starter (₹2,500), Pro (₹6,500), Enterprise (₹16,500)
- ✅ HRMS API integration with module access control
- ✅ Indian Rupee pricing throughout
- ✅ Real-time sync status and history

## Tech Stack

- React 18 (JavaScript)
- Vite
- Tailwind CSS
- Lucide React icons
- Sonner for notifications

## Getting Started

```bash
npm install
npm run dev
```

## Demo Login

Use any valid email and password to access the demo system.

## Architecture

The application runs entirely on JavaScript (.js files) with no TypeScript dependencies.
